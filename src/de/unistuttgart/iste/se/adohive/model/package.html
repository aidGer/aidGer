Provides model interfaces for the use with the Controller.

Name Mappings to database scheme

Table and column names in the database used by AdoHive are named in German, interface names and methods in adohive.model are named in English.
For this reason, here is an easy Mapping between German and English names, the value type (sometimes with range/possible values) and a description where the meaning of a field is not obvious. (GERMAN - ENGLISH)
